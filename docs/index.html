FAQ

  • Q: What is the current status of this?
  • A: It's a public alpha version.
  • Q: What are the next steps?
  • A: Currently the focus lies on fixing bugs and preparing the desktop version for app store releases. The tablet release will follow later on.
  • Q: Wait a second, app store? Will this cost money?
  • A: The desktop and tablet versions will have a certain price tag, yes - I'll try to keep it low though, promised! These versions will also get sprite animation features and save/load to unencrypted files. I plan to give registered users access to a DRM-free version, too.
  • Q: What about the browser version?
  • A: The current plan is that the browser version will remain free, but will not feature sprite animation and only save to encrypted files.
  • Q: I found a bug or miss a feature, what should I do?
  • A: Please report it at the issue tracker on Github or write an email.
